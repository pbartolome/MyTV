# MyTV

An AppleTV aggregator of TV video providers. Currently supported channels:

 * RTVE (Highlights)
 * Atresmedia - Antena3/La Sexta (Highlights)
 * TVE1, TVE2, Antena3, La Sexta, Telemadrid, Canal 24h, Teledeporte (Live)
